# demo-resume-builder

Demo project for job application.

## Quick guide

Requirements.

- Node.js v20. Recommend way to setup is [NVM](https://github.com/nvm-sh/nvm).
- [pnpm](https://pnpm.io/installation).

```sh
# Run these two commands and check result on web browser
pnpm install
pnpm dev

# Additional command
## Run unit test
pnpm test:unit
## Run linter
pnpm lint
```
